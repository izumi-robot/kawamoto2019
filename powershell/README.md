# PowerShell

PowerShellとは、Microsoftが開発したCUIのシェルです。CUIのシェルとは、文字列でコンピュータと会話する(コンピュータに命令を与える)アプリケーションです。例えて言うなら、LINEでボット(人狼ボットとか)に命令を出すのと似ています。
このフォルダでは、PowerShellの基本的な使い方をまとめておきます。
